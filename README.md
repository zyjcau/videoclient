### 5路视频源组合情况
- 第三路和窗口共享同时只能存在一路
- 第二路的开启顺序必须在第三路或窗口共享之后，否则会导致远端看到得是静止画面
- 最多同时只能4路（第123路+屏幕共享）

### 缺陷：
- 视频上无法叠加透明组件
- 视频时，无法通过WebUI层显示通知
- 第1路和第2路无法同时监听图像帧

### 不同布局控制器方式的差异
- 使用内建布局时
  - 无法在会前设置摄像头分辨率，而自定义布局没有这个问题（setMaxConstraint导致停止执行后续代码）
  - 无法在设置窗口独立渲染摄像头
  - 程序退出时会报异常（非法访问内存，SDK的Sample也是如此）
