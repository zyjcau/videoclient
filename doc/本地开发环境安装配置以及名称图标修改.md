# Win10 本地开发环境安装配置

## 1. 安装 Visual Studio 2022

推荐安装完整 IDE：

```text
Visual Studio 2022 Community / Professional / Enterprise
```

官方下载地址：

```text
https://visualstudio.microsoft.com/zh-hans/vs/
```

## 2. 勾选 Visual Studio 组件

打开 `Visual Studio Installer`，选择 `Visual Studio 2022`，点击 `修改`。

在“工作负载”里勾选：

```text
.NET 桌面开发
```

如果安装的是 Build Tools，勾选：

```text
.NET 桌面生成工具
```

然后切换到“单个组件”，搜索并确认安装以下组件：

```text
.NET Framework 4.7.1 targeting pack
.NET Framework 4.7.1 SDK
NuGet 包管理器
MSBuild
C# and Visual Basic Roslyn compilers
```

如果列表中没有 `.NET Framework 4.7.1 SDK`，可以安装 `.NET Framework 4.7.1 Developer Pack`，它包含 SDK 和 Targeting Pack。

## 3. 手动安装 .NET Framework 4.7.1 Developer Pack

如果 Visual Studio Installer 里没有装上 4.7.1，可以单独安装 Developer Pack。

下载地址：

```text
https://dotnet.microsoft.com/download/dotnet-framework/net471
```


## 4. 安装 Visual C++ 运行库

本项目运行时需要 Microsoft Visual C++ 2015-2022 Redistributable。

建议 x86 和 x64 都安装，官方下载地址：

```text
https://learn.microsoft.com/cpp/windows/latest-supported-vc-redist
```

## 5. 安装 NuGet CLI

下载地址：

```text
https://www.nuget.org/downloads
```

建议放到：

```text
C:\Tools\NuGet\nuget.exe
```

然后把下面目录加入系统或用户 `Path`：

```text
C:\Tools\NuGet
```

## 6. 恢复 NuGet 包

进入项目根目录：

```powershell
cd videoclient 进入项目根目录
```

执行：

```powershell
nuget restore TangoWin.sln
```

如果 `nuget` 命令不可用，使用完整路径：

```powershell
& "C:\Tools\NuGet\nuget.exe" restore TangoWin.sln
```

成功后，项目根目录会出现或补齐：

```text
packages\
```

本项目使用 `packages.config`，NuGet restore 只会下载依赖到 `packages` 目录，不会修改项目代码。


## 7. 用 Visual Studio 编译

打开从github下载的项目videoclient，执行：

```text
生成 -> 生成解决方案
```

成功后点击启动按钮，或者直接运行输出目录下的 exe 文件
